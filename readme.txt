== Description ==

FlexyPress Scroll to Top adds a customizable scroll-to-top button, offering options for color, background color, text, icon, custom CSS, border radius, and more.


== Frequently Asked Questions ==

**Is this plugin require any plugin to be installed?**

No.

**Can I use this plugin with any themes?**

Yes, you can. This plugin works with any themes without any limitation.
